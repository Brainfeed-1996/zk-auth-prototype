# Security Policy

## Supported Versions
This project is actively maintained on the default branch.

## Reporting a Vulnerability
Please open a **private security advisory** on GitHub when possible.
If unavailable, open an issue titled security: <short-description> with minimal reproduction details.

We target:
- Initial triage: **48 hours**
- Mitigation plan: **7 days** for high/critical issues
- Coordinated disclosure after fix publication

Do not publish exploit details before a patch is available.
